What’s New in EVE Viewer

Version 1.37.0 (2022-05-12)

  • [Feature-#2767] The Engine now supports a simulation mode, i. e. no connections via channel access are made but the measurement sequence is simulated by constants such that a “normal” looking data file is created for validation up front. See the Engine View description on how to enable/disable simulation mode and how it is visualized.
  • [Bug-#6516] fixed a bug where a wrong engine status was shown when a nested scan was paused by the user.

For older versions, see the What’s New Archive.
