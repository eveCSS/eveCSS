Scan Module View

In the Scan Module view you set the properties for a scan module.

No of Measurements

The No of Measurements is a given number of measurements taken for each motor position.

Trigger Delay

The Trigger Delay is a time in seconds for waiting after positioning between the Prescan action and the triggering of the detector channels.

Settle Time

The Settle Time is a time in seconds for waiting after the first positioning in the scan module. The time is waiting between the Prescan action and the triggering of the detector channels.

Manual Trigger

With Manual Trigger you define, whether triggering of Motors and Detectors should be confirmed.

Actions

In the Actions item you see in a table with six tabs the selected options for Motor Axes, Detector Channels, Prescan, Postscan, Positioning and Plot.

Motor Axes

In the Motor Axes tab you define, which axis are used in the scan module. A description of Motor Axis is here

Detector Channels

In the Detector Channels tab you define, which channels are used in the scan module. A description of Detector Channel is here

Prescan

In the Prescan tab you define, which prescan action will be done before the scan module is started.
P. Die Prescan Aktionen werden ausgeführt, nachdem die erste Position des Scan Modules angefahren wurde. Ob die Prescan-Aktion auf eine Antwort wartet oder nur einfach abgeschickt wird, hängt von ihrer Vereinbarung im xml File ab (method=„GETPUTCB” oder =„GETPUT”)

Postscan

In the Postscan tab you define, which postscan action will be started after the scan module is done.
P. Ob die Postscan-Aktion auf eine Antwort wartet oder nur einfach abgeschickt wird, hängt von ihrer Vereinbarung im xml File ab (method=„GETPUTCB” oder =„GETPUT”)

Positioning

In the Positioning tab you define, which type of positionig will be started after the scan module and/or prescan actions will be done. The Positioning move axis to calculated positions.

Plot

In the Plot tab you define, how many plot windows you will seen in the scan module. A description of the Plot Window is here

Events

In the Events Tab you choose which Events you need in the scan module. The description of events an which event types are available in the scan module view you found here
