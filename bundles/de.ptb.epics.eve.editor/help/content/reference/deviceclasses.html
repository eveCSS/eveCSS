Device Classes (and their influence)

Each device (motor, motor axis, detector, detector channel and pre-/postscan device) could be assigned a class name. Depending on that class is the location in the context menu of the scan module view the device appears in.

Device Class Name Location
Motor C child of root element „C”1
— appears in the root list1
Motor Axis C child of root element „C”
— child of its parent (Motor)1
Detector C child of root element „C”2
— appears in the root list2
Detector Channel C child of root element „C”
— child of its parent (Detector)2
(Pre-/Postscan) Device C child of root element „C”
— appears in the root list

1 If a Motor only has one child element (Axis) the child is pulled up a level and the motor disappears.

2 If a Detector only has one child element (Channel) the child is pulled up a level and the detector disappears.
