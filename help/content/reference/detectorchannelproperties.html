Detector Channel Properties

Property DescriptionDomainMandatory
AverageThe (arithmetic) mean is calculated from Average measurements. (if omitted no average is calculated.)false
Max. Deviation (%)The deviation (%) of the first two measured values must be less or equal the given value. If that constraint is not fulfilled the first measurement is dropped and a new one is taken until the condition is met. Then the Average calculation begins. (If omitted the tolerance check always validates.)floatfalse
MinimumIf (and only if) the first measured value is ≥ Minimum a tolerance check is conducted, otherwise only one value is measured. (if omitted no tolerance check is done.)floatfalse
Max.AttemptsIf the tolerance check fails Max.Attempts times the lastly calculated mean is used (and a warning is given). (If omitted Max.Attempts = ∞ is assumed, if Max.Attempts < Average then Max.Attempts = Average is assumed.)false
Confirm Trigger manualSets whether triggering should be confirmed by user interaction.{true, false}—

Example 1

Average = ∅, Max. Deviation (%) = ∅, Minimum = ∅, Max. Attempts = ∅


Average omitted ⇒ no average is calculated.

Example 2

Average = 3, Max. Deviation (%) = ∅, Minimum = ∅, Max. Attempts = ∅


Minimum omitted ⇒ Average is calculated from three subsequent measurements.

Example 3

Average = 3, Max. Deviation (%) = ∅, Minimum = 0.01, Max. Attempts = ∅


measured value: 0.00215 ⇒ Average = 0.00215 (first measured value < Minimum).

Example 4

Average = 3, Max. Deviation (%) = ∅, Minimum = 0.0001, Max. Attempts = ∅


measured values: {0.00197, 0.00199, 0.00153} ⇒ Average = 0.00183

Example 5

Average = 3, Max. Deviation (%) = 0.1, Minimum = 0.0001, Max. Attempts = ∅


measured values: {0.00247, 0.00243, 0.00237, 0.00244, 0.00214, 0.00214, 0.00217}

⇒ Average = 0.00215

Example 6

Average = 3, Max. Deviation (%) = 0.1, Minimum = 0.1, Max. Attempts = ∅


measured value: 0.00216 ⇒ Average = 0.00216 (first measured value < Minimum).

Example 7

Average = 3, Max. Deviation (%) = 0.01, Minimum = 0.0001, Max. Attempts = 3


measured values: {0.00203, 0.00210, 0.00212} ⇒ Average = 0.00212 (Max. Attempts reached).
