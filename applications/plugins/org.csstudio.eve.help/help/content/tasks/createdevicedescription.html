Create a Device Description

To avoid typing XML tags over and over again a device description is not created within XML itself but with the aid of a shell script called eveBuild. eveBuild uses a macro substitution tool to create device definition files for EVE from simple user-created substitution files.

To create a device description (XML) from your substitution file (xsubst) named ‚mydevices.xsubst’ just type

eveBuild mydevices.xsubst

By omitting the file argument the default file is taken. In this case the directory the command is called from has to be the main directory of your test site, e.g. /messung/test. The name of the default file then would be test.xsubst. In detail, the script does

  • translate the given xsubst into XML
  • replaces occurences of ‚/’ characters in id tags with ‚%’ (EPICS PV name restrictions)
  • validates the resulting XML file with the current XML schema
  • searches for duplicate id tags

More information on the syntax of substitution files is given here.
