Manipulate Properties of a Device

In order to change the properties of a motor axes or detector channel follow these steps:

  1. Select the scan module containing the device of interest by (left-)clicking on it.
  2. Select the device you want to change (left click) in the appropriate table (motor axes / detector channels table) in the scan module view.
  3. Whether you selected a motor axes or detector channel the corresponding view (Motor Axis View or Detector Channel View) will show the current settings of the device.
  4. Now you can change the properties of the device.

To get more information about the effects of these settings consult motor axis properties and detector channel properties .
